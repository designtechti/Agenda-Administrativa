# Agenda Administrativa

	Este repositório é uma agenda administrativa escrita em PHP, onde na página inicial mostra cada compromisso por ondem de data/horário e faz com que todos os compromissos mudem de cor de acordo com a data.

	Por exemplo, se o evento ocorrernos próximos dias (ou nas próximas horas), ficará com a cor verde. Se ocorrer no mesmo dia em que se esta consultando a agenda (dia atual), a cor mudará para amarelo. Já se o evento estiver passado, ficará com a cor vermelha.
